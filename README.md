# TI-84 Style Calculator App

This project is a simple web-based calculator inspired by the TI-84 graphing calculator experience. It includes a bright, calculator-style interface with number buttons, arithmetic operators, memory helpers, decimal input, and keyboard support.

## Features

- TI-84-inspired layout and color scheme
- Basic arithmetic operations: addition, subtraction, multiplication, and division
- Decimal input and backspace support
- Memory functions: clear, recall, add, and subtract
- Keyboard support for quick input

## Files

- index.html: calculator structure and buttons
- styles.css: visual styling for the TI-84-like appearance
- script.js: calculator logic and button behavior

## How to Run

Open index.html in your browser to use the calculator.

If you prefer a local server, you can run:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000 in your browser.

## Example Usage

- Enter numbers with the on-screen buttons or your keyboard
- Press an operator to start a calculation
- Press = to see the result
- Use MC, MR, M+, and M- for simple memory support

## Notes

This is a lightweight demo app designed for learning and styling practice rather than full scientific calculator functionality.
